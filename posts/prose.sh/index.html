prose.sh

Uma plataforma de blog para hackers

Texto de 23 de agosto de 2024

Depois do Bear Blog e do Mataroa Blog, indico o prose.sh.

Ele segue a mesma “pegada” minimalista das outras 2 plataformas, mas se define como uma plataforma de blog para hackers.

Isso porque permite publicar posts diretamente da linha de comando, via rsync, scp ou sftp.

Além disso, permite o upload de imagens, uma página 404 personalizada e usar seus próprios estilos.

Se você vive na tela preta e acha chateação demais precisar de uma interface web para publicar, dê uma olhada no prose.sh. 🙂

Crie a Porra de um Blog é uma adaptação de Start a Fucking Blog por Guilherme Magalhães Gall.
